このソフトは学内の資料の手順3から4までを自動で行うものです。
このソフトを使う前に、Virtualboxとvagrantが導入されている事を確認してください。
また、フォルダの場所はどこでも大丈夫です。(一応資料ではこのファイルはCドライブ直下にある事になってます。)
kotamanegiはこのソフトを使った結果生じた損害等について責任を取りません。自己責任でお願いします。


手順表:

--------------Windows篇---------------
1. Windows_Preparationを開きます。
2. 黒い画面が出るので自動的に消えるまで待ちます。
   途中で選択肢が現れた場合は全て「はい」を選択してください。万一「いいえ」を押した場合はもう一度最初からやり直してください。
   黒い画面がすぐに閉じた場合はvagrantがインストールされていない可能性が高いです。
   vagrantをインストールしている事を再確認してください。
3. Windows_MakeLinux を開きます。
4. 黒い画面が出ます。しばらく待った後、最終的にUbuntuと表示されたログイン画面が表示されれば成功です。
   起動後は必ずLinuxを右上のバッテリーマークのところからシャットダウン後VirtualBoxのほうから起動してください。再起動するまでネットワークにつながりません。
5. Linuxがインストールできた事が確認出来たらWindows_MakeLinuxを消してください。万が一もう一度開いてしまうとデータがすべて消えます。

注意:スリープモードに移行しないように気を付けてください。万一スリープモードになってしまった場合は黒い画面を手動で消してもう一度1からやり直してください。
